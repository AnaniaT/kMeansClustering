This project was completed as part of the weekly graded practice labs in the three-course series "Machine Learning Specialization" by Stanford & DeepLearning.AI on Coursera in December 2023. The uploaded files are solely for demonstrating the tasks completed during the project.

### Getting Started

To get a local copy up and running set up a python virtual environment and install the required modules from the requirements file using pip.
  ```sh
  pip install -r requirements.txt
  ```
